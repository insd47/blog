# Blog

Next.js App Router blog boilerplate with shadcn/ui and MDX.

## Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- shadcn/ui b0 preset
- `@next/mdx`

## Getting Started

```bash
pnpm install
pnpm dev
```

Open http://localhost:3000.

## MDX

MDX routes are enabled through `@next/mdx` in `next.config.ts`.
Global MDX component overrides live in `src/mdx-components.tsx`.
