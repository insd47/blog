이 글은 `content/posts` 디렉터리에 글을 추가하는 방식을 보여주는 예시입니다.

## 글 작성 규칙

각 글은 YAML frontmatter와 MDX 본문으로 구성됩니다.

```mdx
---
title: "글 제목"
description: "목록과 메타데이터에 사용할 짧은 설명"
publishedAt: "2026-06-14"
tags:
  - Next.js
draft: false
---
```

`draft: true`인 글은 프로덕션 빌드에서 목록과 정적 경로에 포함되지 않습니다.

## MDX 사용

본문에서는 일반 Markdown 문법과 React 컴포넌트를 함께 사용할 수 있습니다.
공통 MDX 컴포넌트는 `src/mdx-components.tsx`에서 관리합니다.
